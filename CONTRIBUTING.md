# Contribution Guidelines

Thanks for taking the time to contribute :+1:

Please note that this project is released with a
[Contributor Code of Conduct](CODE_OF_CONDUCT.md). By participating in this
project you agree to abide by its terms.

## How can I contribute?

If you're aware of an awesome publicly shared WebMCP contribution, experiment, article, or other helpful resource that you believe would be interesting to the WebMCP community, please create a new issue or submit a pull request. It will be considered for inclusion into the curated list of awesome things related to the WebMCP proposal. For the time being, we only accept contributions to the README.md file that contains links to external resources. Please follow the convention established in that file.

## What is out of scope for this repo?

Any content that violates [Contributor Code of Conduct](CODE_OF_CONDUCT.md) is out of scope.

Material technical contributions to the proposed WebMCP specification happen in a separate repository. To contribute to the specification development, you are expected to [join the Community Group](https://webmachinelearning.github.io/community/#join).
